# AI 時代的新聞工作

2026 海外特派員營分享簡報。

**線上觀看：** https://u0401006.github.io/ai-era-news-2026/

## 使用

- 瀏覽器直接開啟 `presentation.html`
- 鍵盤：`← → ↑ ↓ Space` 翻頁、`F` 全螢幕、`P` 顯示講者筆記
- 右上角切換深／淺色模式
- 底部 hover 出現縮圖導覽

## 結構

- Part 1　AI 大浪來了，媒體受衝擊
- Part 2　你的識讀能力幾級？
- Part 3　新聞倫理的新課題：這樣可以嗎？
- Part 4　中央社媒體實驗室的 AI 應用
- 結語　新聞的未來？

## 重建

```bash
python3 build2.py
```

編輯 `build2.py` 的 `SLIDES` 列表後重跑即可。
